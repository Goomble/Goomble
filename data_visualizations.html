Interactive Map

San Francisco's Tree Maintance by District

Project Description

Map of San Francisco (bounded by supervisor districts) that displays the location of each occurence of tree maintance.

Project Implementation

Here's the link to the running map

Dataset Overview

The data was collected and posted by SF Open Data as one of their reports, and it stores all the "SF311 cases created since 7/1/2008 with location information". I used D3 and Javascript to retrieve the location information from a "point" category, and then I placed a circle to District Map of San Francisco for each point. Out of all the data provided, I chose 5 categoris to display in tooltip: "Source" ("how the service request was made"), "Address", "Neighborhood", "Category" ("The Human readable name of the specific service request type"), "Status" ("The current status of the service request").

Map Interpretation

This map displays all the places where tree maintance has occured in San Franisco, since July 1st, 2008, where every "source" ("how the service request was made") is encoded by color. The visualization is highly interactive, allowing the user to filter the data by both source and district. By hovering over each district on the map, we can see its number, and if we click on the district, we'll see all of the places where tree maintance occured in the selected district. We can also filter the data by "source": hovering over the color legend on the left of the graph gives us a preview of all the locations where the tree maintance was requested with the selected source; clicking the color legend will draw the points on the map. Warning: To avoid confusion, you can only apply one filter (either district or source) at a time. If you want to change the filter, hide all the points first by clicking "Hide All" button located below the color legend.

Learining Outcomes

By playing with this map, we can depict a lot of trends from it. For instance, we can see that the sources for most of the tree maintance cases were "Voice In" and "Open311", which makes sense, since those are resources specifically dedicated for the city's maintance, and everyone can leave the request for it. However, the majority (2 out of 3) of occurences of tree maintance in Treasure Island were reported by Integrated Agency, meaning that the community there is not actively reporting, when a tree maintance is needed. Moreover, filtering the data by district tells us that most of the tree maintance occured in districts 5 and 8. These districts are not the biggest on the map, but they might have a larger quantity of trees.
